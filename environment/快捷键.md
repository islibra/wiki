# IDEA

+ 导入包：`Alt + Enter`
+ 自动生成构造方法或Getter/Setter：`Alt + Insert`
+ 删除整行：`Ctrl + Y`
+ 格式化代码：`Ctrl + Alt + L`

+ 单行注释/取消注释：`Ctrl+/`
+ 块注释/取消注释：`Ctrl+Shift+/`

+ 查找类：`Ctrl + N`
+ 查找类方法：`Ctrl + F12` 或 `Alt + 7`
+ 高亮当前变量：`Ctrl+Shift+F7`
+ 在高亮中查找：`F3`或`Shift+F3`
+ 取消高亮：`Esc`
+ 全局搜索：`Ctrl + Shift + F`

+ 查看类继承关系：`Ctrl + H`
+ 查看接口的实现类：`Ctrl + Alt + B` 或 `Ctrl + Alt + 左键`
+ 查看maven依赖关系：打开pom文件，`Ctrl + Alt + Shift + U`

+ 显示当前所在方法：`Alt + Q`
+ 方法跳转：`Alt + 上` 或 `Alt + 下`


# Windows

+ 锁定计算机：`Win + L`

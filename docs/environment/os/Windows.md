## 重启&关机

1. 开始菜单，输入`cmd`回车进入命令行；
2. `shutdown -r -t 5`
    - `-r`表示restart，重启；
    - `-t`表示time，重启操作执行前等待的时间，秒为单位。该命令表示5秒后计算机将重新启动，可以自行修改时间，0表示马上执行重启操作；
3. `shutdown -s -t 0`
    - `-s`表示shutdown，关机；

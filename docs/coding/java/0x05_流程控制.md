# 0x05_流程控制

## switch case break

```java
switch (2) {
    case 1:
        System.out.println(1);
    case 2:
        System.out.println(2);
        // 如果不加break, 将一直往下执行, 输出2 3 default
        break;
    case 3:
        System.out.println(3);
    default:
        System.out.println("default");
}
```

# 快速排序

以第一个元素作为基准值，把数组划分为小于基准值和大于基准值两个子数组，递归。  
- 平均复杂度O(nlogn)
- 最坏复杂度O(n2)

> **Tips:** 合并排序O(nlogn)，但常量大于快排。

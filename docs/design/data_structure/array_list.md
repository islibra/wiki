# array_list

## 数组

- 大小固定, 无法动态扩展(扩容)
- 无法在任意位置插入删除, 需移动后面所有元素, **顺序写O(n)**
- {==可直接读取指定索引的元素==}, **随机读O(1)**

!!! example "适用于查找排序"

## 链表

- 每个元素存储了下一个元素的地址, 可{==动态扩展==}(扩容)
- {==可在任意位置插入删除==}, **随机写O(1)**
- 无法直接读取中间某个元素, 查找时需要遍历整个链表, **顺序读O(n)**

!!! example "适用于实现队列"

## 十字链表

以数组保存头指针, 以链表保存数据

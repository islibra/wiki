## 广度优先搜索

!!! abstract "breadth-first search(BFS), 解决 **无权图** 的最短路径问题, 计算A到B有几跳"

使用 **队列** 广度优先搜索，先将第一层放入队列，弹出第一个元素判断，再将该顶点所有邻居放入队列，若队列为空，即未找到元素。

- 记录已检查过的节点，判断之前校验是否已检查过, 防止出现环
- 每层分一个队列，以记录跳数
- 队列中的元素为链表，连接到起点

!!! note "时间复杂度为 **O(V+E)**"

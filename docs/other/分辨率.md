# 分辨率

> 分辨率的尺寸单位：像素每英寸PPI(pixels per inch)

## 视频

- 480x320, 640x480, 标清
- 720P=1280x720, 高清(HD)
- 1080i隔行扫描
- 1080P逐行扫描=1920x1080=200w, 全高清Full high-definition(FHD), 需要屏幕支持200w像素
- 4K=3840(4096)x2160=884w, 超高清ultra high-definition(UHD), 需要屏幕支持800w像素
- 8K=7680x4320=3317w

## 摄像头

参考mate9:

- 前置: 3264x2448 = 800w
- 后置: 5120x3840 = 2000w(RGB) + 1200w(黑白)

## 屏幕

参考mate9:

1920x1080=200w
